Let's talk about streams

In Unix and Linux, you can use a pipe

|

to redirect the stdout of one of your programs to the stdin of another.

This is incredibly important and useful, because it allows us to essentially connect multiple programs together without the programs needing to know about the internals of each other.

streams are essentially a special type of node eventemitter

if you can handle using an eventemitter, you can handle using a stream
